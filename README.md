# Energy Quest – Intelligent Search Agent Game

This project implements a classical AI agent using informed and uninformed search
algorithms to navigate a grid world while managing limited energy resources.

## Algorithms
- Breadth-First Search
- A* Search with custom heuristics

## Concepts
- Goal-based agent
- Utility-aware decision making
- Heuristic design
- Performance comparison

## How to Run
1. Install Python 3.10+
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`
